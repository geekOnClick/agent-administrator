# Возможности
Этот агент предназначен для работы с файлами (счетами) формата xlsx/xls/pdf/doc/docx или простым общением с локальной моделью gemma4:e4b-8k используя провайдер Ollama.
При работе со счетами модель использует tool `process_bills` для обработки документов и подсчета итоговой суммы к оплате.
Для работы с файлами требуется указать путь к папке или конкретному файлу.

# Установка
Для запуска агента выполните
- npm install
- npm run start
- в CLI режиме выберите интересующий вас режим работы (документы или общение)
